# CafeFlow User Flow

## Customer QR Ordering

1. Pelanggan scan QR di meja.
2. Sistem membuka halaman menu dengan token meja.
3. Pelanggan memilih menu dan membuka cart.
4. Pelanggan membayar lewat QRIS/cashier flow.
5. Kasir menerima status pembayaran.
6. Barista menerima queue realtime.
7. Barista menandai pesanan siap diantar.
8. Pelayan menerima daftar siap antar.
9. Pelayan mengantar pesanan dan menandai selesai.

## Role Flow

- Super Admin: pantau semua cabang, transaksi, pendapatan, user, dan aktivitas sistem.
- Admin: kelola operasional cabang, staff, menu, meja, dan laporan harian.
- Kasir: terima order masuk, validasi pembayaran, cetak struk.
- Barista: lihat queue, ubah status sedang dibuat dan siap antar.
- Pelayan: ambil pesanan siap antar, update status pengantaran.
