# CafeFlow UI/UX Design System

## Style

Modern Cafe Management System dengan visual minimalis, premium, clean UI, dan dukungan dark mode serta light mode.

## Palette

- Coffee Brown: `#6f4e37`
- Deep Espresso: `#3d2416`
- Cream: `#f4ead8`
- Clean White: `#ffffff`
- Elegant Black: `#171412`
- Gold Accent: `#c79a4b`

## Typography

- Primary: Inter atau system sans-serif.
- Heading: besar, padat, tanpa letter spacing negatif.
- Dashboard text: ringkas, mudah dipindai, dan tidak memakai hero-scale type di panel kecil.

## Components

- Buttons: primary coffee, gold CTA, secondary surface, icon-only theme toggle.
- Cards: radius 8px, border halus, shadow ringan.
- Widgets: metric card, order row, branch monitoring, chart panel, activity feed.
- Dialogs: pembayaran, pindah meja, detail pesanan, konfirmasi hapus, sukses order.
- Navigation: sidebar role dashboard, sticky landing navbar, bottom nav pelanggan mobile.

## Responsive Rules

- Mobile: pelanggan memakai frame mobile, dashboard stack satu kolom, nav ringkas.
- Tablet: feature grid dua kolom, dashboard panel tetap scan-friendly.
- Desktop: landing hero dua kolom, dashboard sidebar + content, grid widget empat kolom.
